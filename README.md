# ⏳ TimeSpend

แอพ PWA ที่แปลง "การใช้เงิน" ให้เป็น "เวลาชีวิต" — แรงบันดาลใจจากหนัง *In Time*

ทุกครั้งที่จ่ายเงิน คุณจ่ายด้วยเวลาชีวิต และนาฬิกาก็ **ไหลลดเรียลไทม์** เหมือนในหนัง

## แนวคิด

- **นาฬิกาชีวิตไหลลดเรียลไทม์** (1 วินาที/วินาที) — เติมเวลาเมื่อมีรายรับ, กัดเวลาเพิ่มเมื่อใช้จ่าย
- **เปิด/ปิดได้** — สลับเป็น "นาฬิกานิ่ง" (ลดเฉพาะตอนใช้เงิน) ได้ในตั้งค่า
- นาฬิกา **เปลี่ยนสี** เขียว → เหลือง → แดง ตามเวลาที่เหลือ
- ทำงาน **offline 100%** — เก็บข้อมูลใน IndexedDB + localStorage ไม่มีเซิร์ฟเวอร์

## อัตราแลกเปลี่ยน (เงิน → เวลา)

อิง **ค่ากินอยู่/เดือน** เพื่อให้ "การใช้ชีวิต" เผาเวลาจริง 1:1 พอดี:
```
อัตรา R = ค่ากินอยู่ ÷ 730.5 ชม.จริง/เดือน   (เงินกี่บาท = ชีวิต 1 ชม.)
เงินเดือนซื้อเวลา = เงินเดือน ÷ R
```
- เงินเดือน **> ค่ากินอยู่** → ได้เวลา > 1 เดือน = ส่วนเกินสะสม (ยิ่งประหยัดยิ่งมีบัฟเฟอร์)
- ใช้จ่ายฟุ่มเฟือย → กัดเวลาเพิ่ม → ใช้เกินตัว = หมดก่อนเงินเดือนออก
- รองรับทั้งโหมด **เงินเดือน** และ **ค่าแรง/ชม.** (คำนวณรายได้/เดือนให้)

## รันโปรเจกต์

```bash
npm install
npm run dev        # เปิด dev server
npm run build      # build production (สร้าง PWA + service worker)
npm run preview    # ทดสอบ build บนมือถือในวง LAN
```

## โครงสร้าง

```
src/
  lib/
    time.js        # แปลงเงิน<->เวลา + จัดรูปแบบ
    db.js          # IndexedDB (transactions)
    settings.js    # localStorage (ตั้งค่า)
  useApp.js        # state hook หลัก
  components/
    ClockScreen.jsx   # หน้านาฬิกา (หน้าหลัก)
    AmountSheet.jsx   # แผ่นกรอกจำนวนเงิน (ใช้/เติม)
    RateForm.jsx      # ฟอร์มตั้งค่าอัตราแลก
    Onboarding.jsx    # หน้าตั้งค่าครั้งแรก
    Settings.jsx      # ตั้งค่า + export/reset
  App.jsx
  main.jsx
```
